## Bharat Textile Net Dataset

## 1. Purpose and Origin

The **Bharat Textile Net Dataset** has been assembled exclusively for **non-commercial educational and research purposes** as part of the course:

**CSE3292 — Deep Learning for Computer Vision**

This dataset supports research in **traditional textile pattern recognition and Indian clothing classification** using machine learning and computer vision techniques.

The images in this dataset were collected from **publicly accessible internet sources**, including:

- E-commerce fashion platforms
- Handloom and textile documentation portals
- Cultural and museum archives
- Educational institutions
- Public image search platforms

The dataset was created as part of an academic project by the student team listed in the project documentation.
